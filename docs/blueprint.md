# **App Name**: BankAdviceFlow

## Core Features:

- Payee Registry: A secure interface to add, view, and manage individual payee records, capturing essential personal and banking details.
- Payee Search & Selection: Dynamic dropdowns with search capabilities (by ID or name) for quickly selecting payees during advice creation.
- Bulk Payee Uploader: Tool for importing multiple payee records simultaneously via file upload (e.g., CSV), streamlining initial setup or large updates.
- Bank Advice Composer: An intuitive form for compiling new bank advice documents, enabling users to add multiple payees, specify individual net payments, and input overarching advice details (subject, debit account).
- Advice Narrative AI Assistant: A generative AI tool to intelligently suggest or pre-fill descriptive text for the bank advice based on purpose or context.
- Bank Advice Document Generation: Generate print-ready PDF documents formatted to resemble a professional bank advice, complete with payee breakdowns and total amounts.
- Advice History & Tracking: A dashboard to view all previously issued bank advices, providing status updates and the option to re-export or archive.
- Automated Advice Number Generation: Automatically generate unique bank advice numbers, incorporating the financial year for systematic tracking and record-keeping.

## Style Guidelines:

- A light color scheme to convey professionalism and clarity. The primary color is a deep navy blue (#2D4E86) to instill trust and seriousness. The background is a very light greyish-blue (#ECF0F4) for readability. The accent color is a fresh cyan-blue (#44C0E0) for interactive elements and highlights.
- The font 'Inter' (sans-serif) will be used for both headlines and body text to ensure a modern, neutral, and highly legible experience across all forms and documents.
- Utilize minimalistic, clean line icons that directly relate to financial concepts, transactions, and document management, prioritizing clarity and ease of recognition.
- Employ a structured and organized layout with clear headings, consistent spacing, and efficient use of tabular formats for payee lists and transaction breakdowns. The design will be desktop-first, given the data entry and document generation nature of the application.
- Subtle, non-distracting animations will provide visual feedback for user actions such as form submissions, data loading, or successful document generation, enhancing the professional user experience.
